# College Images

Please add your college images here:

1. **manipal-university.jpg** - Image of Manipal University, Jaipur (the building with golden dome)
2. **iimt-university.jpg** - Image of IIMT University, Meerut (campus view)

## How to Add Images:

1. Save the images from your device
2. Rename them as:
   - `manipal-university.jpg` (or .png)
   - `iimt-university.jpg` (or .png)
3. Place them in this `public/assets/` folder
4. The portfolio will automatically display them in the Education section

## Supported Formats:
- .jpg / .jpeg
- .png
- .webp

## Image Tips:
- Recommended size: 800x600px or larger
- Keep file size under 500KB for faster loading
- Images will be automatically optimized and displayed beautifully
